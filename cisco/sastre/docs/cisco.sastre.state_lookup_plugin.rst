state -- State commands. Faster and up-to-date synced state data.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++


.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This state lookup returns list of SD-WAN devices from vManage, contains multiple arguments with connection and filter details to retrieve state device data. Following parameters must be configured in ansible inventor file - ansible_host - ansible_user - ansible_password - vmanage_port - tenant - timeout

Examples
--------

.. code-block:: yaml+jinja

    
        - name: Fetch all devices state data
          debug:
            msg: "{{ query('cisco.sastre.state', cmd=['bfd','sessions'])}}"
            
        - name: Fetch devices state data with filter arguments
          debug:
            msg: "{{ query('cisco.sastre.state', cmd=['bfd','sessions'], detail=True, site='100', regex='.*', reachable=true, system_ip='10.1.0.2')}}"
